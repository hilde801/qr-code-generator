# About QR Code Generator (qr-code-generator)

A simple QR code generator web app.

# Getting Started

Running qr-code-generator

```sh
# Using Yarn
yarn dev

# Using NPM
npm run dev
```

Building qr-code-generator

```sh
# Using Yarn
yarn build

# Using NPM
npm run build
```
