## Getting Started

### Prerequisites

- NodeJS
- Yarn (optional)

### Building/Running QR Code Generator

```bash
# Building and running QR Code Generator using NPM
$ npm install
$ npm run dev

# Building and running QR Code Generator using Yarn
$ yarn
$ yarn dev
```
